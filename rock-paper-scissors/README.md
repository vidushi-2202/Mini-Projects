# ✊ Rock Paper Scissors

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Play against computer
- Random move generation
- Score tracking
- Interactive UI

## 🧠 Concepts Used
- DOM manipulation
- Event handling
- Random number generation

## ▶️ How to Run
Open index.html in your browser
