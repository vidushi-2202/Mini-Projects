# 🎮 Tic Tac Toe Game

A fully functional Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This project demonstrates DOM manipulation, game logic, and event handling.

---

## 🚀 Features
- ✅ 2-player gameplay
- ✅ Turn-based logic (X & O)
- ✅ Win detection system
- ✅ Draw detection
- ✅ Reset / New Game option
- ✅ Responsive UI

---

## 🧠 Concepts Used
- DOM Manipulation
- Event Listeners
- Arrays & Game Logic
- Conditional Statements
